# React Native AI - Groq Provider Skills

Claude Code skills for `@umituz/react-native-ai-groq-provider` - Groq AI integration with ultra-fast inference.

## Installation

```bash
npx skills add /Users/umituz/Desktop/github/umituz/apps/mobile/npm-packages/react-native-ai-groq-provider/skills/SKILL.md -g
```

## Usage

Say: **"Setup Groq AI in my app"**

## Features

- **Ultra-fast inference** - Up to 10x faster than competitors
- **Multiple models** - LLaMA, Mixtral, Gemma
- **Streaming support** - Real-time text generation
- **Free tier** - 30 requests/min, 14,400/day
- **Open-source models** - No vendor lock-in
